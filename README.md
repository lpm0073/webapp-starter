## Bootstrap Me
```
# Prerequisites
brew install libjpeg libpng

npm install --global gulp@next jshint
npm install --global yo
npm install --global generator-webapp
npm install --global bower

# Web App generator
yo webapp
bower install visionmedia/page.js
bower install hover
```
